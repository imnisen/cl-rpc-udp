* Cl-Rpc-Udp 

** Usage
This is a raw and flawed rpc over udp framework now.

** Installation
1. Clone to somewhere quicklisp can find, such as ~~/quicklisp/local-projects~
2. Run (ql:quickload :cl-rpc-udp) in lisp repl to load it
